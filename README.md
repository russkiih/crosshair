# Crosshair

A customizable crosshair overlay for DayZ and other games where an in-game crosshair is not provided.

## Features

- Always-on-top crosshair overlay
- Multiple crosshair styles (dot, cross, circle, dot_circle)
- Adjustable size and thickness
- Customizable color
- Global toggle hotkey (default: F8)
- Settings dashboard for easy configuration
- Automatic startup option
- Tray icon for quick access

## Installation

1. Download the latest release from the [Releases](https://github.com/yourusername/crosshair/releases) page
2. Extract the files to your preferred location
3. Run the application by double-clicking `Crosshair.exe`

## Usage

- The crosshair will appear at the center of your screen
- Open the settings dashboard from the tray icon
- Customize your crosshair's appearance and behavior
- Toggle the crosshair on/off with the configured hotkey (default: F8)

## Development

This application is built with Electron.

### Prerequisites

- Node.js (v14 or newer)
- npm

### Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/crosshair.git
cd crosshair

# Install dependencies
npm install

# Start the application
npm start
```

### Building

```bash
# Install electron-builder
npm install -g electron-builder

# Build for your platform
electron-builder
```
